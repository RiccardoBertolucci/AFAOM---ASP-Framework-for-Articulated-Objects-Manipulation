### results

File Name: time_[NUMBER OF LINKS]_[GRANULARITY]

Each file contains the times and the number of actions neccessary to succesfully solve a problem. For each of the five instances 
we have for a couple [NUMBER OF LINKS] - [GRANULARITY] there is aline composed as follow:

clingo_[NUMBER OF LINKS]_[GRANULARITY]-n[INSTANCE NUMBER],t,a

where:

- t is the total planning time computed with the "pyrunlim.py" script
- a is the number of actions
